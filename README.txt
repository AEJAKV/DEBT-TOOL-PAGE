CANADIAN DEBT PAYOFF TOOL — LANDING PAGE

FILES
- index.html: Landing page
- Canadian_Debt_Payoff_Tool.xlsx: Downloadable Excel workbook
- canadian_debt_tool_preview.png: Workbook dashboard preview

HOW TO USE
1. Keep all three files in the same folder.
2. Upload the folder contents to your website host.
3. Make sure the file names are not changed, or update the links in index.html.
4. Open index.html in a browser to test the page.

QUICK HOSTING OPTIONS
- Upload the files to your existing website hosting.
- Drag the folder into Netlify Drop.
- Add the files to a GitHub repository and enable GitHub Pages.
- Deploy the folder through Vercel as a static site.
